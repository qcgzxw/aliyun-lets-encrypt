# 阿里云WAF证书自动上传
该工具用于自动上传服务器申请的Let's Encrypt或其他证书。

推荐使用ACME脚本的hook功能，acme更新域名同时执行该脚本上传证书。
## 使用方式
### 下载二进制文件
### 手动执行
### 配置acme
### 子用户创建
### 子用户添加权限
